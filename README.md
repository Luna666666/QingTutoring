# QingTutoring

## QingTutoring是什么?
一个基于Objective-C语言开发的高内聚低耦合的iOS开发项目
## QingTutoring有哪些功能？

* 辅导班`辅导班`功能
* 课程`课程`功能
* 发现`发现`功能
* 我的`我的`功能
